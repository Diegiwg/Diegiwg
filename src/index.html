Conhecimentos

Iniciei no mundo da programação com o Python, escrevendo um Webscrap para raspar algumas imagens de um site de mangas online, para ler em meu celular, sem internet.

Minha história com Javascript está primordialmente relacionada a modificação de sites que uso/usava, utilizando essa linguagem para construir scripts para o Tampermonkey, com mais diversos objetivos. Nesse âmbito, o maior trabalho que fiz, foi recentemente, modificar um aplicativo web feito em angular, para adicionar uma função, que automatizou um trabalho diário que eu fazia.

A primeira vez que tentei usar HTML foi para construir uma biblioteca, modificando o tamplate do Blogspot, para compartilhar alguns livros que tinha.

Sempre fiquei intrigado com o quão bonito alguns sites são, então comecei a investigar seus codigos, e eventualmente descobri o CSS. Fiquei receoso com essa linguagem de estilo, mas depois que se compreende seu funcionamento, ela da muita liberdade para o desenvolver.
